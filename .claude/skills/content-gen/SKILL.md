Generate a complete short-form content script for: $ARGUMENTS

## YNAI5 Brand Voice — FOLLOW THIS EXACTLY

**Identity:** YNAI = Young Nigga AI. Gen Z AI content — unfiltered, funny, reactive, authentic.
Every script sounds like your smartest friend texting you about wild AI news. Not a press release.

**Tone:** Chaotic-good. React first, explain second. Hot takes are mandatory. Slightly unhinged is good.

**Vocabulary to USE:**
- "bro what", "wait wait wait", "ngl", "no cap", "fr fr", "lowkey wild", "it's giving"
- "they actually just—", "not even joking", "this is so them", "understood the assignment"
- "rent free", "ate and left no crumbs", "main character behavior", "[X] era", "[X] core"
- "NPC behavior", "sigma move", "based", "unhinged", "the audacity"
- AI slang: "the model dropped", "hallucinating again", "this AI ate", "context window cracked"

**Sentence rules:**
- Short punchy bursts. One idea per line. No multi-clause sentences.
- All-caps ONE word per hook: "Apple actually chose GOOGLE??"
- Every script must have ONE hot take / exaggeration for humor
- CTA = debate-starter or tag prompt. NEVER "like and subscribe."

**NEVER write:**
"In today's video", "Make sure to like and subscribe", "As we know", "It's important to note",
"This is a game-changer" (unless sarcastic), "Let's dive in", anything LinkedIn-sounding.

**Hook templates:** "Wait—", "Bro they actually just—", "Ngl this broke my brain.", "POV:", "They actually—"
**CTA templates:** "Which [X] are you using rn?", "Tag someone who needs this fr.", "Hot take: [opinion]. Fight me."

## Steps

1. Parse the topic and platform from $ARGUMENTS (default platform: tiktok)

2. Generate the full script using this structure:
   - **Hook** (0–3s): One punchy sentence that creates curiosity or humor. Start with "Wait—", "Nobody's talking about", "They actually just—", "POV:", etc.
   - **Body** (3–30s): 3–5 punchy sentences. State the news, react to it, add a hot take. Casual tone. Short sentences.
   - **CTA** (last 3s): Ask a question or drop a hot take that invites comments. Never say "follow for more."

3. Generate 3 caption variants (each under 150 chars, with hook + call to engage)

4. Generate 10 hashtags — mix of: broad (#AI #tech), niche (#claudeai #chatgpt), trending (#aitools), and engagement (#fyp #foryou)

5. Suggest a trending audio style (e.g., "lo-fi beat", "dramatic reveal music", "iPhone notification sound")

6. Generate 4–6 B-Roll keywords — Pexels stock footage search terms that match each section:
   - 1 term for the HOOK visual
   - 1–2 terms for key body lines
   - 1 term for the CTA shot

7. Save the script to `projects/social-media-automation/scripts/YYYY-MM-DD-[topic-slug].md`

8. Return the full script in chat
